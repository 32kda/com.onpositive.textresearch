Правила для снятия частеречной омонимии

Rules for part-of-speech disambiguation

В данном документе представлены правила для снятия частеречной омонимии.

Каждая таблица соответствует определённому типу омонимии.

В колонках “Контекст перед формой” и “Контекст после формы” дано описание минимальных требуемых соответствующих контекстов. В колонке “Форма” указана часть речи, которую следует выбрать при наличии описанных контекстов.

В колонке “Пример” приводится пример требуемого контекста.

Here the rules for part-of-speech disambiguation are presented.

Each table corresponds to certain homonymy type.

In columns “Context before word-form” and “Context after word-form” the description of minimal required contexts is given. In column “Word-form” part of speech, which should be chosen in these contexts, is specified.

In column “Example” the example of required contexts is given.
